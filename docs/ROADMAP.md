# Roadmap

## Phase 0: 起動確認

- [ ] `python -m local_strategist_voice` で画面が開く
- [ ] `config.json` がない場合に分かりやすいエラーが出る
- [ ] LM Studio の Local Server に接続できる
- [ ] AivisSpeech / VOICEVOX 互換APIに接続できる

## Phase 1: MVP完成

- [ ] テキスト入力を送信できる
- [ ] ローカルLLMから返答が返る
- [ ] 返答が音声として再生される
- [ ] Markdownログが `logs/YYYY-MM-DD.md` に保存される
- [ ] `persona.md` の変更が応答に反映される

## Phase 2: 使いやすさ改善

- [ ] 接続テストボタンを追加する
- [ ] 音声再生のオン/オフ切替を追加する
- [ ] 話者IDを画面から変更できるようにする
- [ ] LLMモデル名を画面から変更できるようにする
- [ ] エラー文を初心者向けに翻訳して表示する

## Phase 3: 音声入力

- [ ] マイク録音ボタンを追加する
- [ ] Whisper / faster-whisper / ローカルSTT候補を調査する
- [ ] 音声入力 → LLM → 音声応答の流れを作る

## Phase 4: 参謀化

- [ ] 会話ログから作戦メモを自動生成する
- [ ] Obsidian向けMarkdown出力を整える
- [ ] プロジェクト別ログ保存に対応する
- [ ] RAGによる過去ログ参照を追加する

## Phase 5: 画面監視・常駐化

初期版では扱わない。MVPが安定してから実装する。

- [ ] 常駐トレイアプリ化
- [ ] 画面OCR / スクリーンショット解析
- [ ] 危険操作を避ける権限制御
- [ ] 自動実行ではなく、提案・確認ベースの支援
